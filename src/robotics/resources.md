### How to Add a PDF

**For team members:** Write your content in Markdown and send it to Mr. Reveal, who will add it to the website.

**For Mr. Reveal:** To add a new PDF to this page:

1. Place the PDF file in `/src/assets/pdfs/` directory
2. Add a link in this markdown file using this format:

```markdown
- [PDF Title](/assets/pdfs/filename.pdf) - Brief description of contents
```

Example:
```markdown
- [Drivetrain Assembly Guide](/assets/pdfs/drivetrain-guide.pdf) - How to assemble our mecanum drive system
```
